Schemas v1 - Validação de Dados

Schema: midia_item

Schema para validação de itens de mídia coletados (notícias, artigos, entrevistas, vídeos, posts).

Campos Obrigatórios:
id candidato fonte url tipo titulo data_publicacao texto
Campos Opcionais:
resumo score_credibilidade origem_coleta
Regras de Validação:
  • texto: Mínimo 400 caracteres
  • titulo: Entre 10-200 caracteres
  • score_credibilidade: Valor entre 0-1
  • data_publicacao: Formato ISO 8601
  • fonte: Deve estar na whitelist

Schema: pesquisa_item

Schema para validação de pesquisas eleitorais coletadas de institutos de pesquisa.

Campos Obrigatórios:
id instituto local data_campo amostra margem metodologia cenario resultado
Campos Opcionais:
fonte_url confiabilidade_baixa
Regras de Validação:
  • amostra: Entre 100-10.000 entrevistados
  • margem: Entre 1-10 pontos percentuais
  • resultado: Array com candidato e percentual
  • instituto: Deve estar na lista autorizada
  • local: Distrito Federal, DF ou Brasília
